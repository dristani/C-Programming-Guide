2. Data Types

Understanding different data types in C programming

Introduction to Data Types

Data types in C specify the type of data that a variable can store. C provides several basic data types, and understanding them is crucial for writing efficient programs.

Basic Data Types

Data Type Size (bytes) Range Format Specifier
char 1 -128 to 127 %c
int 4 -2,147,483,648 to 2,147,483,647 %d
float 4 1.2E-38 to 3.4E+38 %f
double 8 2.3E-308 to 1.7E+308 %lf

Example Program

#include <stdio.h>

int main() {
    char grade = 'A';
    int age = 25;
    float height = 5.9;
    double salary = 75000.50;

    printf("Grade: %c\n", grade);
    printf("Age: %d\n", age);
    printf("Height: %.1f\n", height);
    printf("Salary: %.2lf\n", salary);

    return 0;
}

Code Explanation

  • char grade = 'A'; - Declares a character variable
  • int age = 25; - Declares an integer variable
  • float height = 5.9; - Declares a floating-point variable
  • double salary = 75000.50; - Declares a double-precision floating-point variable
  • printf() statements demonstrate different format specifiers for each data type

Type Modifiers

C provides type modifiers that can alter the range and storage size of basic data types:

  • signed - Allows both positive and negative values (default)
  • unsigned - Allows only positive values
  • short - Reduces the size of the data type
  • long - Increases the size of the data type

Tips and Best Practices

  • Choose the appropriate data type based on the range of values needed
  • Use unsigned types when negative values are not needed
  • Be careful with floating-point comparisons due to precision issues
  • Use sizeof() operator to check the size of data types on your system
  • Consider memory usage when choosing between float and double
