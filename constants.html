4. Constants

Understanding constants in C programming

Introduction to Constants

Constants in C are fixed values that cannot be modified during program execution. They are used to store values that remain unchanged throughout the program's lifecycle.

Types of Constants

C supports several types of constants:

  • Integer constants (e.g., 10, -5, 0)
  • Floating-point constants (e.g., 3.14, -0.5)
  • Character constants (e.g., 'A', '1', '$')
  • String constants (e.g., "Hello, World!")
  • Symbolic constants (defined using #define or const)

Example Program

#include <stdio.h>

// Using #define for symbolic constants
#define PI 3.14159
#define MAX_SIZE 100
#define GREETING "Hello, World!"

int main() {
    // Using const keyword for constants
    const int DAYS_IN_WEEK = 7;
    const float GRAVITY = 9.81;
    const char NEWLINE = '\n';
    
    // Integer constants
    printf("Integer constant: %d\n", 42);
    
    // Floating-point constants
    printf("PI value: %.5f\n", PI);
    printf("Gravity: %.2f m/sÂ²\n", GRAVITY);
    
    // Character constants
    printf("Character constant: %c\n", 'A');
    printf("Newline character: %c", NEWLINE);
    
    // String constants
    printf("%s\n", GREETING);
    
    // Using constants in calculations
    int radius = 5;
    float area = PI * radius * radius;
    printf("Area of circle with radius %d: %.2f\n", radius, area);
    
    // Array with constant size
    int numbers[MAX_SIZE];
    printf("Array size: %d\n", MAX_SIZE);
    
    return 0;
}

Code Explanation

  • #define PI 3.14159 - Defines a symbolic constant using preprocessor directive
  • const int DAYS_IN_WEEK = 7; - Defines a constant using const keyword
  • printf("Integer constant: %d\n", 42); - Uses a literal integer constant
  • float area = PI * radius * radius; - Uses symbolic constant in calculation
  • int numbers[MAX_SIZE]; - Uses symbolic constant for array size

Constant Declaration Rules

  • Symbolic constants are typically written in uppercase
  • Constants must be initialized when declared
  • Constants cannot be modified after declaration
  • Character constants are enclosed in single quotes
  • String constants are enclosed in double quotes

Frequently Asked Questions

Q: What is the difference between #define and const?

A: #define is a preprocessor directive that replaces text before compilation, while const creates a read-only variable with type checking.

Q: Can I use variables in #define constants?

A: No, #define constants are replaced with their literal values during preprocessing, before variable values are known.

Q: Why use constants instead of variables?

A: Constants provide better code readability, prevent accidental modifications, and can help the compiler optimize the code.

Q: Can I use expressions in #define constants?

A: Yes, but you should use parentheses to ensure proper evaluation, e.g., #define SQUARE(x) ((x) * (x))
