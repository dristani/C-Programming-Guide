7. Decision Making

Understanding decision making in C programming

Introduction to Decision Making

Decision making in C programming allows you to execute different code blocks based on certain conditions. C provides several decision-making statements that help control the flow of program execution.

Types of Decision Making Statements

Statement Description Syntax
if Executes a block of code if a condition is true if (condition) { /* code */ }
if-else Executes one block if condition is true, another if false if (condition) { /* code */ } else { /* code */ }
if-else if-else Tests multiple conditions in sequence if (condition1) { /* code */ } else if (condition2) { /* code */ } else { /* code */ }
switch Selects one of many code blocks to execute switch (expression) { case value1: /* code */ break; default: /* code */ }
nested if if statement inside another if statement if (condition1) { if (condition2) { /* code */ } }

Example Program

#include <stdio.h>

int main() {
    int number;
    char grade;
    
    // Simple if statement
    printf("Enter a number: ");
    scanf("%d", &number);
    if (number > 0) {
        printf("The number is positive\n");
    }
    
    // if-else statement
    if (number % 2 == 0) {
        printf("The number is even\n");
    } else {
        printf("The number is odd\n");
    }
    
    // if-else if-else statement
    if (number < 0) {
        printf("The number is negative\n");
    } else if (number == 0) {
        printf("The number is zero\n");
    } else {
        printf("The number is positive\n");
    }
    
    // switch statement
    printf("Enter a grade (A-F): ");
    scanf(" %c", &grade);
    switch (grade) {
        case 'A':
            printf("Excellent!\n");
            break;
        case 'B':
            printf("Good!\n");
            break;
        case 'C':
            printf("Average\n");
            break;
        case 'D':
            printf("Below Average\n");
            break;
        case 'F':
            printf("Failed\n");
            break;
        default:
            printf("Invalid grade\n");
    }
    
    // Nested if statement
    if (number > 0) {
        if (number % 2 == 0) {
            printf("The number is positive and even\n");
        } else {
            printf("The number is positive and odd\n");
        }
    }
    
    return 0;
}

Code Explanation

  • if (number > 0) - Checks if the number is positive
  • if (number % 2 == 0) - Checks if the number is even
  • else if (number == 0) - Checks if the number is zero
  • switch (grade) - Evaluates different grade cases
  • case 'A': - Handles the case when grade is 'A'
  • default: - Handles any case not explicitly defined

Decision Making Best Practices

  • Always use curly braces {} for if statements, even for single-line blocks
  • Use meaningful condition expressions
  • Keep nested if statements to a minimum (preferably no more than 3 levels)
  • Use switch statements when comparing a single variable against multiple values
  • Always include a default case in switch statements
  • Use logical operators (&&, ||) to combine conditions when needed

Frequently Asked Questions

Q: What is the difference between if-else and switch statements?

A: if-else is more flexible and can handle complex conditions, while switch is more efficient for comparing a single variable against multiple constant values.

Q: When should I use nested if statements?

A: Use nested if statements when you need to check multiple conditions that are dependent on each other. However, try to keep the nesting level minimal for better readability.

Q: Why is the break statement important in switch cases?

A: The break statement prevents fall-through to the next case. Without it, execution would continue to the next case, which is usually not desired.

Q: Can I use floating-point numbers in switch statements?

A: No, switch statements can only work with integer types (int, char) and enumerated types. For floating-point comparisons, use if-else statements.
