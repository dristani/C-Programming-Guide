18. Input and Output

Understanding input and output operations in C programming

Introduction to Input and Output

Input and Output (I/O) operations in C are performed using standard library functions. The stdio.h header file provides functions for reading input from the user and displaying output to the screen. C supports both formatted and unformatted I/O operations.

I/O Operations

Operation Function Description Example
Formatted Output printf() Displays formatted output printf("Value: %d", num);
Formatted Input scanf() Reads formatted input scanf("%d", &num);
Character Output putchar() Displays a single character putchar('A');
Character Input getchar() Reads a single character char c = getchar();
String Output puts() Displays a string puts("Hello");
String Input gets() Reads a string gets(str);

Format Specifiers

Specifier Data Type Description
%d int Decimal integer
%f float Floating point
%c char Character
%s char[] String
%lf double Double precision
%p void* Pointer address

Example Program

#include <stdio.h>

int main() {
    // Basic I/O operations
    int num;
    float fnum;
    char ch;
    char str[50];
    
    // Formatted input
    printf("Enter an integer: ");
    scanf("%d", &num);
    
    printf("Enter a float: ");
    scanf("%f", &fnum);
    
    printf("Enter a character: ");
    scanf(" %c", &ch);  // Note the space before %c
    
    printf("Enter a string: ");
    scanf("%s", str);
    
    // Formatted output
    printf("\nYou entered:\n");
    printf("Integer: %d\n", num);
    printf("Float: %.2f\n", fnum);
    printf("Character: %c\n", ch);
    printf("String: %s\n", str);
    
    // Character I/O
    printf("\nEnter another character: ");
    ch = getchar();  // Clears the newline
    ch = getchar();  // Gets the actual character
    printf("You entered: ");
    putchar(ch);
    printf("\n");
    
    // String I/O
    printf("\nEnter another string: ");
    scanf(" %[^\n]", str);  // Reads until newline
    printf("You entered: ");
    puts(str);
    
    // Advanced formatting
    printf("\nFormatted Output Examples:\n");
    printf("Integer with width: %5d\n", num);
    printf("Float with precision: %.3f\n", fnum);
    printf("Hexadecimal: %x\n", num);
    printf("Octal: %o\n", num);
    printf("Scientific notation: %e\n", fnum);
    
    return 0;
}

Code Explanation

  • scanf("%d", &num); - Reads an integer from input
  • scanf(" %c", &ch); - Reads a character (note the space)
  • scanf("%s", str); - Reads a string
  • printf("%.2f", fnum); - Prints float with 2 decimal places
  • ch = getchar(); - Reads a single character
  • puts(str); - Prints a string with newline

I/O Characteristics

  • Standard I/O is buffered
  • Formatted I/O uses format specifiers
  • Input functions return values indicating success/failure
  • Output functions return number of characters printed
  • I/O operations can be redirected
  • File I/O uses similar functions
  • Error handling is important for robust I/O

I/O Best Practices

  • Always check return values of input functions
  • Use appropriate format specifiers
  • Handle buffer overflow in string input
  • Clear input buffer when needed
  • Use meaningful prompts for user input
  • Format output for readability
  • Handle errors gracefully

Frequently Asked Questions

Q: What is the difference between puts() and printf()?

A: puts() automatically adds a newline character at the end and can only print strings, while printf() is more flexible and can print formatted output of various types but doesn't automatically add a newline.

Q: Why do we need to use & with scanf() for variables?

A: The & operator is used to get the address of a variable. scanf() needs the address of variables to store the input values directly in memory. This is why we use & for basic data types, but not for arrays (like strings) which are already pointers.

Q: How can I clear the input buffer?

A: You can clear the input buffer using while ((getchar()) != '\n'); or fflush(stdin); (though fflush(stdin) is not standard C). This is often needed after using scanf() to remove any remaining characters in the buffer.

Q: What are the common format specifiers in C?

A: Common format specifiers include %d for integers, %f for floats, %c for characters, %s for strings, %lf for doubles, %p for pointers, %x for hexadecimal, and %o for octal. You can also use modifiers like %.2f for precision or %5d for width.
