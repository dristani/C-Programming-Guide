8. Loops

Understanding loops in C programming

Introduction to Loops

Loops in C programming allow you to execute a block of code repeatedly. They are essential for performing repetitive tasks efficiently. C provides several types of loops to handle different looping requirements.

Types of Loops

Loop Type Description Syntax
while Executes code while condition is true while (condition) { /* code */ }
do-while Executes code at least once, then repeats while condition is true do { /* code */ } while (condition);
for Executes code a specific number of times for (init; condition; increment) { /* code */ }

Example Program

#include <stdio.h>

int main() {
    int i, j, n;
    
    // while loop
    printf("While loop example:\n");
    i = 1;
    while (i <= 5) {
        printf("%d ", i);
        i++;
    }
    printf("\n\n");
    
    // do-while loop
    printf("Do-while loop example:\n");
    i = 1;
    do {
        printf("%d ", i);
        i++;
    } while (i <= 5);
    printf("\n\n");
    
    // for loop
    printf("For loop example:\n");
    for (i = 1; i <= 5; i++) {
        printf("%d ", i);
    }
    printf("\n\n");
    
    // Nested loops
    printf("Nested loops example (Multiplication table):\n");
    for (i = 1; i <= 5; i++) {
        for (j = 1; j <= 5; j++) {
            printf("%d\t", i * j);
        }
        printf("\n");
    }
    
    // break and continue
    printf("\nBreak and continue example:\n");
    for (i = 1; i <= 10; i++) {
        if (i == 5) {
            break;  // Exit the loop
        }
        if (i % 2 == 0) {
            continue;  // Skip even numbers
        }
        printf("%d ", i);
    }
    printf("\n");
    
    return 0;
}

Code Explanation

  • while (i <= 5) - Executes while i is less than or equal to 5
  • do { ... } while (i <= 5); - Executes at least once, then checks condition
  • for (i = 1; i <= 5; i++) - Initializes i to 1, runs while i â‰¤ 5, increments i after each iteration
  • for (i = 1; i <= 5; i++) { for (j = 1; j <= 5; j++) } - Nested loops for creating a multiplication table
  • break; - Exits the loop immediately
  • continue; - Skips the current iteration and continues with the next

Loop Control Statements

  • break: Terminates the loop immediately
  • continue: Skips the current iteration and continues with the next
  • goto: Jumps to a labeled statement (use with caution)

Loop Best Practices

  • Use for loops when you know the exact number of iterations
  • Use while loops when the number of iterations is unknown
  • Use do-while when you need to execute the loop at least once
  • Avoid infinite loops by ensuring the loop condition will eventually become false
  • Keep loop bodies small and focused
  • Use meaningful variable names for loop counters
  • Be careful with nested loops as they can quickly become complex

Frequently Asked Questions

Q: What is the difference between while and do-while loops?

A: A while loop checks the condition before executing the code, while a do-while loop executes the code at least once before checking the condition.

Q: When should I use a for loop instead of a while loop?

A: Use a for loop when you know the exact number of iterations, and use a while loop when the number of iterations depends on a condition that may change during execution.

Q: What is an infinite loop and how can I avoid it?

A: An infinite loop occurs when the loop condition never becomes false. To avoid it, ensure your loop condition will eventually be met and that variables in the condition are properly updated.

Q: When should I use break and continue statements?

A: Use break to exit a loop when a specific condition is met, and use continue to skip the current iteration when certain conditions are true. However, use them sparingly as they can make code harder to follow.
