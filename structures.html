14. Structures

Understanding structures in C programming

Introduction to Structures

Structures in C are user-defined data types that allow you to combine data items of different kinds. They are used to create complex data types that can group related variables together.

Structure Operations

Operation Description Example
Declaration Defining a structure type struct Student { ... };
Initialization Creating structure variables struct Student s1;
Access Accessing structure members s1.roll_no
Pointer Access Accessing via structure pointer ptr->roll_no
Array of Structures Creating multiple structures struct Student class[50];
Nested Structures Structures within structures struct Address { ... };

Example Program

#include <stdio.h>
#include <string.h>

// Structure definition
struct Student {
    int roll_no;
    char name[50];
    float marks;
    struct Address {
        char street[50];
        char city[30];
        int pin;
    } address;
};

int main() {
    // Structure variable declaration and initialization
    struct Student s1 = {
        .roll_no = 101,
        .marks = 85.5,
        .address = {
            .street = "123 Main St",
            .city = "New York",
            .pin = 10001
        }
    };
    strcpy(s1.name, "John Doe");
    
    // Accessing structure members
    printf("Student Details:\n");
    printf("Roll No: %d\n", s1.roll_no);
    printf("Name: %s\n", s1.name);
    printf("Marks: %.2f\n", s1.marks);
    printf("Address: %s, %s - %d\n", 
           s1.address.street, 
           s1.address.city, 
           s1.address.pin);
    
    // Array of structures
    struct Student class[3];
    for(int i = 0; i < 3; i++) {
        printf("\nEnter details for student %d:\n", i+1);
        printf("Roll No: ");
        scanf("%d", &class[i].roll_no);
        printf("Name: ");
        scanf("%s", class[i].name);
        printf("Marks: ");
        scanf("%f", &class[i].marks);
    }
    
    // Structure pointer
    struct Student *ptr = &s1;
    printf("\nAccessing via pointer:\n");
    printf("Name: %s\n", ptr->name);
    printf("Marks: %.2f\n", ptr->marks);
    
    return 0;
}

Code Explanation

  • struct Student { ... }; - Structure definition with nested Address structure
  • struct Student s1 = { ... }; - Structure variable initialization
  • s1.roll_no - Accessing structure members using dot operator
  • struct Student class[3] - Array of structures
  • ptr->name - Accessing structure members using pointer
  • s1.address.street - Accessing nested structure members

Structure Characteristics

  • Structures can contain different data types
  • Structure members are accessed using dot operator
  • Structure pointers use arrow operator (->)
  • Structures can be nested
  • Structures can be passed to functions
  • Structures can be returned from functions
  • Structures can be used to create complex data types

Structure Best Practices

  • Use meaningful names for structures and members
  • Initialize structure variables properly
  • Use typedef for cleaner code
  • Pass large structures by pointer
  • Keep related data together in structures
  • Document structure purpose and usage
  • Consider memory alignment and padding

Frequently Asked Questions

Q: What is the difference between structure and union?

A: In a structure, each member has its own memory location, while in a union, all members share the same memory location. Structures are used when you need to store different types of data together, while unions are used when you need to store only one type of data at a time.

Q: How do I pass a structure to a function?

A: You can pass structures to functions by value (copy) or by reference (pointer). For large structures, it's better to pass by pointer to avoid copying the entire structure. Use the arrow operator (->) to access members when using pointers.

Q: What is structure padding?

A: Structure padding is the process of adding extra bytes between structure members to align them according to the system's memory alignment requirements. This can affect the size of the structure and should be considered when working with memory-sensitive applications.

Q: How do I use typedef with structures?

A: typedef can be used to create an alias for a structure type. For example: typedef struct Student Student; allows you to declare variables as Student s1; instead of struct Student s1;.
