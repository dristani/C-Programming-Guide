3. Variables

Understanding variables in C programming

Introduction to Variables

Variables are fundamental building blocks in C programming. They are used to store data that can be modified during program execution. Each variable has a specific type, name, and value.

Variable Declaration and Initialization

In C, variables must be declared before they can be used. They can be initialized at the time of declaration or later in the program.

Example Program

Example Program:

#include <stdio.h>

int main() {
    int age = 25;           // Integer variable
    float height = 5.9;     // Float variable
    char grade = 'A';       // Character variable
    double pi = 3.14159;    // Double variable
    
    printf("Age: %d\n", age);
    printf("Height: %.1f\n", height);
    printf("Grade: %c\n", grade);
    printf("Pi: %.5f\n", pi);
    
    return 0;
}

Code Explanation

  • int number; - Declares an integer variable named 'number'
  • float price; - Declares a floating-point variable named 'price'
  • char letter; - Declares a character variable named 'letter'
  • number = 42; - Initializes the 'number' variable with value 42
  • int age = 25; - Declares and initializes an integer variable in one line
  • int x = 10, y = 20, z = 30; - Declares and initializes multiple variables of the same type
  • printf() statements demonstrate how to print different types of variables

Variable Naming Rules

  • Variable names can contain letters, digits, and underscores
  • Must begin with a letter or underscore
  • Cannot be a C keyword (e.g., int, float, return)
  • Are case-sensitive (e.g., 'age' and 'Age' are different variables)
  • Should be meaningful and descriptive

Frequently Asked Questions

Q: What is the difference between declaration and initialization?

A: Declaration tells the compiler about the variable's type and name, while initialization assigns an initial value to the variable.

Q: Can I change a variable's type after declaration?

A: No, once a variable is declared with a specific type, it cannot be changed to another type.

Q: What happens if I use an uninitialized variable?

A: Using an uninitialized variable leads to undefined behavior. The variable may contain garbage values.

Q: Can I declare multiple variables of different types in one line?

A: No, you can only declare multiple variables of the same type in one line.
