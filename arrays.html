11. Arrays

Understanding arrays in C programming

Introduction to Arrays

Arrays in C are collections of elements of the same data type stored in contiguous memory locations. They provide an efficient way to store and access multiple values of the same type.

Array Types

Array Type Description Example
One-dimensional Single row of elements int arr[5];
Two-dimensional Table of elements (rows and columns) int matrix[3][3];
Multi-dimensional Arrays with more than two dimensions int cube[2][3][4];
Character Array Array of characters (string) char str[20];

Example Program

#include <stdio.h>

int main() {
    // One-dimensional array
    int numbers[5] = {1, 2, 3, 4, 5};
    
    // Two-dimensional array
    int matrix[2][3] = {
        {1, 2, 3},
        {4, 5, 6}
    };
    
    // Character array (string)
    char name[] = "C Programming";
    
    // Accessing array elements
    printf("One-dimensional array:\n");
    for(int i = 0; i < 5; i++) {
        printf("numbers[%d] = %d\n", i, numbers[i]);
    }
    
    printf("\nTwo-dimensional array:\n");
    for(int i = 0; i < 2; i++) {
        for(int j = 0; j < 3; j++) {
            printf("matrix[%d][%d] = %d\n", i, j, matrix[i][j]);
        }
    }
    
    printf("\nCharacter array:\n");
    printf("Name: %s\n", name);
    
    // Array operations
    int sum = 0;
    for(int i = 0; i < 5; i++) {
        sum += numbers[i];
    }
    printf("\nSum of numbers array: %d\n", sum);
    
    // Array size
    printf("Size of numbers array: %lu bytes\n", sizeof(numbers));
    printf("Number of elements in numbers array: %lu\n", sizeof(numbers) / sizeof(numbers[0]));
    
    return 0;
}

Code Explanation

  • int numbers[5] = {1, 2, 3, 4, 5}; - Declaration and initialization of a one-dimensional array
  • int matrix[2][3] = {...}; - Declaration and initialization of a two-dimensional array
  • char name[] = "C Programming"; - Declaration and initialization of a character array
  • numbers[i] - Accessing array elements using index
  • sizeof(numbers) - Getting the total size of the array in bytes
  • sizeof(numbers) / sizeof(numbers[0]) - Calculating the number of elements in the array

Array Characteristics

  • Arrays are fixed in size once declared
  • Array elements are stored in contiguous memory locations
  • Array indexing starts from 0
  • Arrays can be initialized at declaration time
  • Arrays can be passed to functions
  • Arrays can be multi-dimensional
  • Character arrays can be used to store strings

Array Best Practices

  • Always initialize arrays when possible
  • Use meaningful array names
  • Be careful with array bounds to avoid buffer overflow
  • Use constants for array sizes
  • Consider using structures for complex data
  • Use appropriate data types for array elements
  • Document array purposes and sizes

Frequently Asked Questions

Q: What is the difference between an array and a pointer?

A: An array is a collection of elements of the same type, while a pointer is a variable that stores a memory address. Arrays can decay into pointers in certain contexts, but they are not the same thing.

Q: Can I change the size of an array after declaration?

A: No, arrays in C have a fixed size that cannot be changed after declaration. If you need a resizable array, consider using dynamic memory allocation with malloc() and realloc().

Q: How do I pass an array to a function?

A: Arrays are passed to functions by reference (as a pointer to the first element). You can pass them using the array name, and you should also pass the size of the array as a separate parameter.

Q: What is array decay?

A: Array decay is the process where an array is automatically converted to a pointer to its first element when used in most expressions. This happens in function calls, assignments, and other contexts.
