1. Basic Syntax

Understanding the fundamental structure of C programs

Introduction to C Syntax

C programming language follows a specific syntax that defines how programs are written and structured. Understanding this syntax is crucial for writing correct and efficient C programs.

Basic Structure of a C Program

Every C program follows a basic structure that includes:

  • Preprocessor directives
  • Main function
  • Program statements
  • Comments

Example Program

#include <stdio.h>

// This is a comment
int main() {
    printf("Hello, World!\n");
    return 0;
}

Code Explanation

  • #include <stdio.h> - Includes the standard input/output header file
  • // This is a comment - Single-line comment
  • int main() - Main function declaration
  • printf("Hello, World!\n"); - Output statement
  • return 0; - Indicates successful program execution

Key Syntax Rules

  • Every statement ends with a semicolon (;)
  • Code blocks are enclosed in curly braces {}
  • Header files are included using #include directive
  • Comments can be single-line (//) or multi-line (/* */)
  • Function names are followed by parentheses ()

Tips and Best Practices

  • Always include necessary header files
  • Use meaningful variable and function names
  • Properly indent your code for better readability
  • Add comments to explain complex logic
  • Follow consistent coding style
