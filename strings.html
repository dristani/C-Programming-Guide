13. Strings

Understanding strings in C programming

Introduction to Strings

In C, strings are arrays of characters terminated by a null character ('\0'). They are fundamental for text processing and manipulation. Understanding string handling is crucial for many C programming tasks.

String Operations

Operation Description Example
Declaration Creating a string char str[20];
Initialization Assigning value to string char str[] = "Hello";
String Length Getting string length strlen(str);
String Copy Copying one string to another strcpy(dest, src);
String Concatenation Joining two strings strcat(str1, str2);
String Comparison Comparing two strings strcmp(str1, str2);

Example Program

#include <stdio.h>
#include <string.h>

int main() {
    // String declaration and initialization
    char str1[20] = "Hello";
    char str2[20] = "World";
    char str3[20];
    
    // String length
    printf("Length of str1: %lu\n", strlen(str1));
    
    // String copy
    strcpy(str3, str1);
    printf("str3 after copy: %s\n", str3);
    
    // String concatenation
    strcat(str1, " ");
    strcat(str1, str2);
    printf("str1 after concatenation: %s\n", str1);
    
    // String comparison
    if (strcmp(str1, str2) == 0) {
        printf("Strings are equal\n");
    } else {
        printf("Strings are not equal\n");
    }
    
    // String input
    char name[50];
    printf("Enter your name: ");
    fgets(name, 50, stdin);
    name[strlen(name) - 1] = '\0';  // Remove newline
    printf("Hello, %s!\n", name);
    
    // String manipulation
    char text[] = "C Programming";
    printf("Original text: %s\n", text);
    
    // Convert to uppercase
    for(int i = 0; text[i] != '\0'; i++) {
        if(text[i] >= 'a' && text[i] <= 'z') {
            text[i] = text[i] - 32;
        }
    }
    printf("Uppercase text: %s\n", text);
    
    return 0;
}

Code Explanation

  • char str1[20] = "Hello"; - String declaration and initialization
  • strlen(str1) - Returns the length of the string (excluding null terminator)
  • strcpy(str3, str1) - Copies str1 to str3
  • strcat(str1, str2) - Appends str2 to str1
  • strcmp(str1, str2) - Compares two strings (returns 0 if equal)
  • fgets(name, 50, stdin) - Safe way to read string input
  • text[i] = text[i] - 32 - Converting lowercase to uppercase

String Characteristics

  • Strings are null-terminated character arrays
  • String literals are stored in read-only memory
  • String functions are available in string.h header
  • Strings can be manipulated using pointers
  • String length doesn't include the null terminator
  • Strings can be initialized using string literals
  • Strings can be passed to functions as character arrays

String Best Practices

  • Always allocate enough space for strings (including null terminator)
  • Use fgets() instead of gets() for input
  • Check string lengths before operations
  • Use strncpy() instead of strcpy() for safer copying
  • Initialize strings properly
  • Handle string termination correctly
  • Be careful with string literals and memory allocation

Frequently Asked Questions

Q: What is the difference between char array and string?

A: A char array is just an array of characters, while a string is a char array that is null-terminated. All strings are char arrays, but not all char arrays are strings.

Q: Why is gets() unsafe?

A: gets() doesn't check the buffer size, which can lead to buffer overflow. Always use fgets() instead, which allows you to specify the maximum number of characters to read.

Q: How do I compare strings in C?

A: Use strcmp() function from string.h. It returns 0 if strings are equal, negative if first string is less than second, and positive if first string is greater than second.

Q: How do I convert a string to a number?

A: Use atoi() for integers, atof() for floating-point numbers, or strtol()/strtod() for more control over the conversion process.
