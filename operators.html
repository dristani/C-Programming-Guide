6. Operators

Understanding operators in C programming

Introduction to Operators

Operators in C are symbols that tell the compiler to perform specific mathematical or logical operations. C provides a rich set of operators that can be categorized into different types.

Types of Operators

Operator Type Symbols Description
Arithmetic +, -, *, /, % Basic mathematical operations
Relational ==, !=, >, <, >=, <= Comparison operations
Logical &&, ||, ! Boolean operations
Bitwise &, |, ^, ~, <<, >> Bit-level operations
Assignment =, +=, -=, *=, /=, %= Value assignment operations
Increment/Decrement ++, -- Increase or decrease by 1

Example Program

#include <stdio.h>

int main() {
    // Arithmetic operators
    int a = 10, b = 3;
    printf("Arithmetic Operations:\n");
    printf("a + b = %d\n", a + b);
    printf("a - b = %d\n", a - b);
    printf("a * b = %d\n", a * b);
    printf("a / b = %d\n", a / b);
    printf("a %% b = %d\n", a % b);

    // Relational operators
    printf("\nRelational Operations:\n");
    printf("a > b: %d\n", a > b);
    printf("a < b: %d\n", a < b);
    printf("a == b: %d\n", a == b);
    printf("a != b: %d\n", a != b);

    // Logical operators
    int x = 1, y = 0;
    printf("\nLogical Operations:\n");
    printf("x && y: %d\n", x && y);
    printf("x || y: %d\n", x || y);
    printf("!x: %d\n", !x);

    // Bitwise operators
    unsigned int num1 = 5, num2 = 3;
    printf("\nBitwise Operations:\n");
    printf("num1 & num2: %d\n", num1 & num2);
    printf("num1 | num2: %d\n", num1 | num2);
    printf("num1 ^ num2: %d\n", num1 ^ num2);
    printf("~num1: %d\n", ~num1);
    printf("num1 << 1: %d\n", num1 << 1);
    printf("num1 >> 1: %d\n", num1 >> 1);

    // Assignment operators
    int c = 5;
    printf("\nAssignment Operations:\n");
    c += 3; printf("c += 3: %d\n", c);
    c -= 2; printf("c -= 2: %d\n", c);
    c *= 4; printf("c *= 4: %d\n", c);
    c /= 2; printf("c /= 2: %d\n", c);

    // Increment/Decrement operators
    int d = 5;
    printf("\nIncrement/Decrement Operations:\n");
    printf("d++: %d\n", d++);
    printf("++d: %d\n", ++d);
    printf("d--: %d\n", d--);
    printf("--d: %d\n", --d);

    return 0;
}

Code Explanation

  • a + b - Addition operator
  • a % b - Modulus operator (remainder after division)
  • a > b - Greater than comparison
  • x && y - Logical AND operation
  • num1 & num2 - Bitwise AND operation
  • c += 3 - Addition assignment (equivalent to c = c + 3)
  • d++ - Post-increment operator

Operator Precedence

Operators in C follow a specific order of precedence when evaluating expressions:

  1. Parentheses ()
  2. Unary operators (++, --, !, ~, +, -)
  3. Multiplicative (*, /, %)
  4. Additive (+, -)
  5. Shift (<<, >>)
  6. Relational (<, <=, >, >=)
  7. Equality (==, !=)
  8. Bitwise AND (&)
  9. Bitwise XOR (^)
  10. Bitwise OR (|)
  11. Logical AND (&&)
  12. Logical OR (||)
  13. Assignment (=, +=, -=, etc.)

Frequently Asked Questions

Q: What is the difference between ++i and i++?

A: ++i (pre-increment) increments the value before using it, while i++ (post-increment) uses the value first and then increments it.

Q: What is the difference between = and == operators?

A: = is the assignment operator that assigns a value, while == is the equality comparison operator that checks if two values are equal.

Q: When should I use bitwise operators?

A: Bitwise operators are useful for low-level programming, hardware manipulation, and optimizing certain algorithms that work with individual bits.

Q: What is operator precedence?

A: Operator precedence determines the order in which operators are evaluated in an expression. You can use parentheses to override the default precedence.
